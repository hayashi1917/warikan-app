# 旅行者向け支払い管理アプリ

## 使用技術
- **フロントエンド**: HTML, CSS, JavaScript
- **バックエンド**: Python, FastAPI
- **データベース**: MySQL
- **その他**: Docker

## 期間
- 2026年2月～3月（約1カ月）

## 概要
パリに1カ月短期留学した際、大学のプロジェクトとして3人グループで1つのアプリケーションを開発。
旅行をするグループにおいて、お金のやり取りに関するトラブルを減らすことを目的に、会計管理を行うWebアプリケーションを開発。

## 機能
- グループの作成、グループへの参加、ユーザーログイン
- 1度の会計に対して、立て替え者を設定可能
- 各ユーザーごとに個別の請求金額を設定可能
- 支払いに対して、各ユーザーが承認または拒否を選択可能
- 支払い関係者全員が承認した会計のみ、全体の計算に組み込み
- 日本円、米ドル、ユーロ、英ポンドに対応
- 複数の承認済み会計をまとめて、お金の動きが最小になるよう各ユーザーの支払い額・受け取り額を計算

## 技術的な工夫
- 短期間での開発に対応するため、開発効率の高い FastAPI を採用
- 開発前にグループ内でDBのER図を十分に設計したことで、ユーザ登録や支払い承認機能を円滑に実装
- お金のやり取りを最小化するため、貪欲法を用いた精算ロジックを実装
- 外部APIを利用して日本円と各外貨の為替レートを取得し、複数通貨に対応
